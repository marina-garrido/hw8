Age Prediction Based on Name

API Information

Name: Agify API

Description: The Agify API predicts ages based on names.

Website source: https://agify.io/

Options/Parameters Used: The API request is made by providing the name to predict the age for in the query string.

Applications:

  • On a social media platform to provide age predictions for users.
  • In an e-commerce site to personalize product recommendations based on predicted age groups.
